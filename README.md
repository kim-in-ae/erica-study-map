🗺️ ERICA Study Map

한양대학교 ERICA 캠퍼스의 공부하기 좋은 장소를 지도 기반으로 쉽게 확인할 수 있는 웹 서비스이다.

👉배포 주소

https://kim-in-ae.github.io/erica-study-map/

✔️프로젝트 필요성

현재 한양대 에리카 캠퍼스에는 공부할 수 있는 공간이 매우 다양하지만, 각 건물마다 위치도 다르고, 운영 시간이 제각각이며, 조용한 공간/팀플 공간 등 용도도 다르기 때문에 학생들이 본인에게 맞는 공부 공간을 찾기 어렵다는 문제가 있다.
캠퍼스 내 공부 장소 정보를 한눈에 확인할 수 있는 웹 서비스가 필요해 이 프로젝트를 만들게 되었다.

📌프로젝트 중요성
1. 학생들에게 실질적인 도움 제공 : 지도 기반으로 공부 장소의 특징, 위치, 운영 시간을 바로 확인할 수 있다.
2. 학교 공간 활용도 증가 : 잘 알려지지 않은 유용한 공간을 안내함으로 학교 시설의 전체적인 이용률을 고르게 만들 수 있다.
3. 누구나 쉽게 접근 가능한 웹 서비스 : 설치 없이 모바일 또는 PC에서 즉시 접속 가능하고 간단한 구조로 빠르게 로딩되어 사용 편의성이 높다.

✅ 기능 설명

캠퍼스 지도 기반으로 캠퍼스 내 스터디 장소 마커 표시
각 장소 정보 표시 : 이름, 위치, 운영시간, 설명
장소 데이터는 JSON 형식으로 분리되어 있어, 유지 보수 및 추가가 용이

🌐 협업 과정 (역할분담)

역할 1) 지도 렌더링, 마커 표시
역할 2) UI, 스타일링, 페이지 구조(HTML,CSS)
역할 3) 장소 데이터 수집 및 JSON 작성, 문서화, 배포 설정
